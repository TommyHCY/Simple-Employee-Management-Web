# Simple Employee Management Web
 ___
* 這是一個使用Spring Boot和Thymeleaf實現的簡單CRUD (Create, Read, Update, Delete) 員工管理系統範例。您可以透過Web介面添加、查看、更新和刪除員工信息。
### 功能
* 添加員工
* 列出所有員工
* 更新員工信息
* 刪除員工
### 前置條件
* JDK 17 或更高版本
* Maven 3.6+
* H2 資料庫（或其他你選擇的資料庫）
### 使用技術
* Spring Boot：用於構建Java後端應用
* Thymeleaf：用於渲染前端頁面
* H2 database(Local):作為資料庫
* Spring Data JPA：與資料庫進行交互
